Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:

Default Button

In-text link

CSS selector:

.intext-link

.intext-link2

.intext-link3

Sample code:

<a href = "css/main.css" class ="intext-link"> this is an in-text link </a>

Rendered element:

this is an in-text link
this is another in-text link
this is another in-text link

Submit/Reset Buttons

CSS selector:

.submit-style

.reset-style

Sample code:

<input id = "submit-input1" type = "submit" class ="submit-style"> <input id = "reset-input1" type = "reset" class ="reset-style">

Rendered element:

Text Input Field

CSS selector:

.input-style

Sample code:

<label for = "demo-input1"> Search:</label> <input id = "demo-input1" type = "text" class ="input-style">
